#Scriptz

Misc scripts/hacks.

 * [dev](dev) - development-related tools (git, markups, build tools...)
 * [fixes](fixes) - fixes for buggy software
 * [fun](fun) - random useless stuff
 * [launchers](launchers) - .desktop launchers
 * [media](media) - mutimedia tools (conversion, sharing...)
 * [sysadmin](sysadmin) - systems administration tools (file manipulation, network security, configuration, maintenance, monitoring...)
 * [utility](utility) - utilities for daily use (todo list managers, desktop tweaks, converters
 * [web](web) - tools to interact with web services
